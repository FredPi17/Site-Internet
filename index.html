Page d'accueil

Bienvenue sur la page d'accueil du site hébergé par ma Raspberry Pi.

Il permet la diffusion du projet et la visualisation des données météorologiques en local. Quand je dis local

L'objectif de notre projet était de pouvoir récupérer les données de pollution de l'air à partir de capteurs NOx et de les envoyer sur la Raspberry Pi. Ce site Web permet donc de montrer ces données de manière visuelle et parlante pour le client

Ce site est le fruit d'un travail mené sur 3 mois, il a été élaboré par Sophiane et Frédéric.
